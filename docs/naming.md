# Naming

The generic project and its current example server use separate identities:

| Item | Name |
| --- | --- |
| Product/tool | Discord IaC |
| npm package | `discord-iac` |
| Recommended repository | `Discord-IaC` |
| Recommended root folder | `Discord-IaC` |
| Example server profile | `wao-noobs` |
| Current Discord bot | WAO Server Setup |
| Current Discord server | WAO Noobs |

`profiles/wao-noobs` remains unchanged as the logical location of the server-specific desired configuration. WAO-specific names inside that profile and its corresponding live snapshot are legitimate domain data, not stale product branding.

The root folder now uses the recommended `Discord-IaC` name.

Discord IaC is an independent open-source tool name and is not affiliated with Discord.
